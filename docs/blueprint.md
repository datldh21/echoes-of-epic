# **App Name**: Echoes of Epic

## Core Features:

- 3D Virtual Museum: Interactive 3D museum experience showcasing the Hector and Andromache excerpt.
- AI Avatar Generation: Generate 'ancient Facebook' profile avatars with AI based on character descriptions and backstories.
- Interactive Map of Troy: An interactive map with hover effects to reveal details about key locations within the city of Troy.
- Podcast Series: A series of podcasts featuring characters and perspectives related to the Iliad excerpt. AI can be used as a tool to determine if a summary of any episode would enhance user engagement.
- Debate Board: A forum-like system where students can post arguments from different character perspectives and vote on the most convincing one.
- Interactive Minigame: A strategy-based minigame where the user makes choices impacting the defense of Troy, revealing historical insights.
- AI-Powered Art Generation: Automatically generate images from key scene descriptions such as 'Hector removing his helmet for his son'.

## Style Guidelines:

- Primary color: Antique Gold (#D4A373) to evoke a sense of ancient grandeur and value.
- Background color: Midnight Blue (#192841), a desaturated variant of the primary color to evoke a night setting and create contrast for spotlighting elements.
- Accent color: Brick Red (#A73030) analogous to antique gold, for highlighting calls to action.
- Headline font: 'Cinzel' (serif), as requested by the user. Note: currently only Google Fonts are supported.
- Body text: A pairing of 'Lora' (serif) / 'Inter' (sans-serif), as requested by the user. Note: currently only Google Fonts are supported.
- Icons: Use icons resembling ancient Greek pottery art.
- Layout: Mimic museum spotlighting techniques.
- Animation: Add smooth transitions between pages with subtle animations.
